# Intellij 설치

- IntelliJ 는 Java 개발자라면 가장 유명한 IDE중 하나라는 것을 알 것이다. 
- Eclipse STS 는 무료 버젼으로 인기있으며, IntelliJ 는 유료이지만 강력한 기능으로 자바 개발자에게 사랑받는 IDE이다. 

## IntelliJ 설치 

- https://www.jetbrains.com/idea/download/?section=mac 에서 IntelliJ를 다운받고 설치하자. (맥 버젼 주소이며 윈도우 개발자는 윈도우 탭을 클릭해서 설치하자. )

## kotlin 프로그램 생성하기. 

- 코틀린으로 시작하기: https://www.jetbrains.com/help/idea/get-started-with-kotlin.html
- REPL로 코틀린 빠르게 실행하기: https://www.jetbrains.com/help/idea/kotlin-repl.html
- 첫번째 어플리케이션 개발하기: https://www.jetbrains.com/help/idea/create-your-first-kotlin-app.html
- 테스트 드리븐 코틀린 개발하기: https://www.jetbrains.com/help/idea/tdd-with-kotlin.html#write-test-body

- 위 과정을 꼭 따라해보자.
